# Symuality

My corner of the internet. Not a brand. Not a funnel. Just a personal homepage like the ones people used to make.

**Live:** [symuality.com](https://symuality.com)

## What's here

- **/** — Who I am, what I'm into, where to find me
- **/now** — What I'm currently working on, playing, and tinkering with
- **/projects** — Things I've built
- **/posts** — Writing (when I have something to say)
- **/about** — The longer version

## Stack

- [Next.js](https://nextjs.org) (App Router)
- [Keystatic](https://keystatic.com) (file-based CMS for blog posts)
- [Tailwind CSS](https://tailwindcss.com) v4
- Deployed on [Vercel](https://vercel.com)

## Running locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
